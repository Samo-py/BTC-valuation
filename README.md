# Bitcoin Fair Value Model

**Author:** Samuel Tkáč
**Date:** February 24, 2026  

This repository contains a research project on Bitcoin fair value modeling, exploring the estimation of Bitcoin’s “fair price” based on fundamental factors.

---

## Abstract

The question of Bitcoin’s fundamental value remains open. Critics argue that it has no intrinsic value, yet Bitcoin has seen mainstream adoption through ETFs and corporate treasuries. This project develops a model to estimate Bitcoin’s fair price using a supply-side approach based on miner costs and network security.
